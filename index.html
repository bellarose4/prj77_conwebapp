Project on console

Microsoft Visual Studio is an integrated development environment (IDE) from Microsoft. It is used to develop computer programs, as well as websites, web apps, web services and mobile apps.
Visual Studio includes a code editor supporting IntelliSense (the code completion component) as well as code refactoring.

How to set up
Download the Visual Studio bootstrapper file.
Run the bootstrapper file to install the Visual Studio Installer.
Choose workloads.
Install

Console screen
Console enables executing Visual Studio commands and scripts line by line and printing them on the screen.

How to open console screen
Double click on index.html file in folder.
When page opens, right click mouse.
Click on "inspect" option and then click on "console" tab to see console screen.

To type anything into console screen. Use command: console.log("Item to print");
